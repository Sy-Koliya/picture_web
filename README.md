# picture_web

基于 0voice 图床项目，重构后端代码，整体框架从单线程单机架构升级为多线程+分布式服务架构。

---

## 项目概述

- **原项目**：0voice 图床
- **重构目标**：提升并发性能、简化部署、易于水平扩展
- **核心改动**：
  - 多线程处理请求
  - C++20协程处理
  - gRPC 微服务通信

## 架构演进

| 阶段      | 架构                         |
|---------|----------------------------|
| 早期版本   | 单线程同步，单台服务器           |
| 重构后版本 | 多线程 + 分布式微服务架构        |

## 环境依赖

- **C++** 20 及以上  
- **Redis**（`redis-plus-plus` 客户端）  
- **FastDFS**  
- **Nginx** + FastDFS 模块  
- **gRPC**  
- **MySQL**
- **nlohmann**
